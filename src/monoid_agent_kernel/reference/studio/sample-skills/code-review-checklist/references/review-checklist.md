# Compact Review Checklist

- Public contract: inputs, outputs, errors, compatibility.
- State: lifecycle, cleanup, retries, cancellation, concurrency.
- Security: path traversal, command execution, secrets, permissions.
- Data: validation, encoding, size limits, ordering, idempotency.
- Tests: happy path, edge case, regression, failure mode.
- Observability: actionable events, redaction, stable identifiers.
